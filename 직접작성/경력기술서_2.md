Python, FastAPI, Pydantic, FastMCP, Anthropic Claude API / RAG 파이프라인·PV 워크플로·검증 계층·평가 스크립트 자체 구현

•검색 품질(Hit@1·MRR·ContextRecall)·PV 워크플로·검증기 평가 스크립트와 pytest 195케이스를 작성하고 GitHub Actions CI로 회귀 방지, FastAPI 백엔드·웹 챗 UI로 엔드투엔드 데모 구성
