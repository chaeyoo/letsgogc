# 01. 회사 분석 — GC녹십자 / GC그룹

> 자소서 1번(지원 이유)·2번(인재상 연결)·면접의 근거 자료.

---

## 1. 미션 · 비전 · 핵심가치 ⭐ (자소서 필수 인용)

- **미션:** 인류의 건강한 삶에 이바지한다
- **비전:** 건강산업의 글로벌 리더가 된다
- **핵심가치 4가지:**
  | 핵심가치 | 의미 | 내 경험 연결 포인트 |
  |---|---|---|
  | **창의도전** | 두려움 없이 새로운 것에 도전 | ⭐ 마크로젠 레거시 LIMS·낯선 유전자 도메인 정복 / Claude Code 신기술 실무 도입 |
  | **봉사배려** | 사람과 사회를 위한 마음 | 건강·헬스케어 도메인에 커리어를 건 동기 |
  | **정도투명** | 원칙과 정직, 투명함 | 실험실 데이터 정확성·신뢰성을 다룬 경험 |
  | **인간존중** | 사람 중심 | 현업(비개발자)이 쓰기 쉬운 도구를 만든 관점(챗봇 빌더 등) |

- **조직문화:** **"두려움 없는 혁신(fearless innovation)"** — 제품·조직뿐 아니라 직원 개발 방식에서도 혁신 추구

> 📌 이 직무(FDE/AX)와 가장 맞닿는 가치 = **창의도전** + 혁신 문화. 문항 2는 이 가치를 증명하는 방향으로 설계.

---

## 2. 사업 개요
- 혈액제제(면역글로불린 'ALYGLO' 美 FDA 승인)·백신(독감/대상포진) 중심의 국내 대표 제약·바이오
- 희귀질환 치료제, **mRNA 백신 파이프라인**('서울 팬데믹 X 서밋 2026'서 개발전략 공유)
- 지주사 GC(녹십자홀딩스), 계열사(GC녹십자·GC셀·GC케어·GC녹십자의료재단 등)

---

## 3. 최신 AX(AI Transformation) 전략 ⭐ (지원 직무의 배경)
- 전사적 **AX 추진**: 경영지원·R&D·제조 전 영역 혁신
- **Hey.GC 2.0**: 자체 개발 **Agentic AI** 기업 플랫폼. 외부 솔루션 의존 없이 자사 기술·데이터로 구축
- **MCP(Model Context Protocol)** 구조로 이메일·캘린더·회의실·방문관리 등 사내 시스템을 하나의 인터페이스로 통합
- 목표: 단순 반복업무 축소 → 임직원이 전략·의사결정 등 고부가가치 업무에 집중
- 허용준 GC 대표: "AX = 그룹 전반 디지털 역량 결집, 의사결정 정밀도 향상 → 글로벌 헬스케어 경쟁력 강화"

---

## 4. 왜 GC녹십자인가 (내 논리 3줄)
1. **생명과학 전공 + 헬스케어 커리어**(마크로젠→에임메드→KB헬스케어)로 도메인을 몸으로 이해 → 봉사배려·미션 부합
2. **풀스택 + 생성형AI(챗봇빌더) + AI개발도구(Claude Code) 실전 경험** → AX/FDE 직무 즉시 기여 가능
3. GC의 **창의도전·두려움 없는 혁신** 문화 = 새 기술로 현업 문제를 직접 푸는 나의 일하는 방식과 일치
